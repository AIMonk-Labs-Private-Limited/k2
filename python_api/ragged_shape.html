ragged_shape¶

create_ragged_shape2¶

k2.ragged.create_ragged_shape2(row_splits=None, row_ids=None, cached_tot_size=- 1)[source]¶

Construct a RaggedShape from row_ids and/or row_splits vectors. For the overall concepts, please see comments in k2/csrc/utils.h.

Parameters

  • row_splits (Optional[Tensor]) – Optionally, a torch.Tensor with dtype=torch.int32 and one axis

  • row_ids (Optional[Tensor]) – Optionally, a torch.Tensor with dtype=torch.int32 and one axis.

  • cached_tot_size (Optional[int]) – The number of elements (length of row_ids, even if row_ids is not provided); would be identical to the last element of row_splits, but can avoid a GPU to CPU transfer if known.

Return type

Shape

Returns

an instance of _k2.RaggedShape, with ans.num_axes() == 2.

compose_ragged_shapes¶

k2.ragged.compose_ragged_shapes(a, b)[source]¶

Compose two RaggedShape objects where a.num_elements() == b.dim0().

Parameters

  • a (Shape) – First shape to be composed

  • b (Shape) – First shape to be composed; b.dim0() must equal a.num_elements()

Return type

Shape

Returns

an instance of _k2.RaggedShape with ans.num_axes() == a.num_axes() + b.num_axes() - 1. May share metadata with a and b.
