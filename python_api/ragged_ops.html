index¶

k2.ragged.index(src, indexes, need_value_indexes=True, axis=0)[source]¶

Indexing operation on ragged tensor, returns src[indexes], where the elements of indexes are interpreted as indexes into axis axis of src.

Caution

indexes is a 1-D tensor and indexes.dtype == torch.int32.

Parameters

  • src (Union[RaggedArc, RaggedInt, RaggedShape]) – Source ragged tensor or ragged shape to index.

  • axis (int) – The axis to be indexed. Must satisfy 0 <= axis < src.num_axes()

  • indexes (Tensor) – Array of indexes, which will be interpreted as indexes into axis axis of src, i.e. with 0 <= indexes[i] < src.tot_size(axis). Note that if axis is 0, then -1 is also a valid entry in index.

  • need_value_indexes (bool) – If true, it will return a torch.Tensor containing the indexes into src.values() that ans.values() has, as in ans.values() = src.values()[value_indexes].

Returns

  • ans of type _k2.RaggedArc or _k2.RaggedInt (same as the type of src).

  • None if need_value_indexes is False; a 1-D torch.tensor of dtype torch.int32 containing the indexes into src.values() that ans.values() has.

pad¶

k2.ragged.pad(src, mode='constant', value=0)[source]¶

Pad a ragged tensor to a torch tensor.

For example, if src has the following values:

[ [1 2 3] [4] [5 6 7 8] ]

Then it returns a 2-D tensor as follows if value is 0 and mode is constant:

tensor([[1, 2, 3, 0],
        [4, 0, 0, 0],
        [5, 6, 7, 8]])
Parameters

  • src (Union[RaggedInt, RaggedFloat]) – The source ragged tensor, MUST have num_axes() == 2.

  • mode (Literal[‘constant’, ‘replicate’]) – Valid values are: constant, replicate. If it is constant, the given value is used for filling. If it is replicate, the last entry in a list is used for filling. If a list is empty, then the given value is also used for filling.

  • value (Union[int, float]) – The filling value.

Return type

Tensor

Returns

A 2-D torch.Tensor whose dtype is torch.int32 if the input is _k2.RaggedInt tensor or torch.float32 if the input is _k2.RaggedFloat tensor. The tensor returned is on the same device as src.

sum_per_sublist¶

k2.ragged.sum_per_sublist(src, initial_value=0)[source]¶

Return the sum of each sublist.

For example, if src has the following values:

[ [a b] [h j k] [m] ]

Then it returns a 1-D tensor with 3 entries:

  • entry 0: a + b + initial_value

  • entry 1: h + j + k + initial_value

  • entry 2: m + initial_value

Parameters

src (RaggedFloat) – A ragged float tensor. Note that the sum is performed on the last axis.

Return type

Tensor

Returns

Return a 1-D torch.Tensor with dtype torch.float32. Its numel equals to src.tot_size(src.num_axes() - 2).

cat¶

k2.ragged.cat(srcs: List[_k2.ragged.RaggedTensor], axis: int) → _k2.ragged.RaggedTensor¶

Concatenate a list of ragged tensor over a specified axis.

Hint

This is a static method.

Example 1

>>> import k2.ragged as k2r
>>> a = k2r.RaggedTensor([[1], [], [2, 3]])
>>> k2r.cat([a, a], axis=0)
[ [ 1 ] [ ] [ 2 3 ] [ 1 ] [ ] [ 2 3 ] ]
>>> k2r.cat((a, a), axis=1)
[ [ 1 1 ] [ ] [ 2 3 2 3 ] ]

Example 2

>>> import k2.ragged as k2r
>>> a = k2r.RaggedTensor([[1, 3], [], [5, 8], [], [9]])
>>> b = k2r.RaggedTensor([[0], [1, 8], [], [-1], [10]])
>>> c = k2r.cat([a, b], axis=0)
>>> c
[ [ 1 3 ] [ ] [ 5 8 ] [ ] [ 9 ] [ 0 ] [ 1 8 ] [ ] [ -1 ] [ 10 ] ]
>>> c.num_axes
2
>>> d = k2r.cat([a, b], axis=1)
>>> d
[ [ 1 3 0 ] [ 1 8 ] [ 5 8 ] [ -1 ] [ 9 10 ] ]
>>> d.num_axes
2
>>> k2r.RaggedTensor.cat([a, b], axis=1)
[ [ 1 3 0 ] [ 1 8 ] [ 5 8 ] [ -1 ] [ 9 10 ] ]
>>> k2r.cat((b, a), axis=0)
[ [ 0 ] [ 1 8 ] [ ] [ -1 ] [ 10 ] [ 1 3 ] [ ] [ 5 8 ] [ ] [ 9 ] ]
Parameters

  • srcs – A list of ragged tensors to concatenate. They MUST all have the same dtype and on the same device.

  • axis – Only 0 and 1 are supported right now. If it is 1, then srcs[i].dim0 must all have the same value.

Returns

Return a concatenated tensor.

unique_sequences¶

k2.ragged.unique_sequences(src, need_num_repeats=True, need_new2old_indexes=False)[source]¶

Remove repeated sequences.

If src has two axes, this will return the unique sub-lists (in a possibly different order, but without repeats). If src has 3 axes, it will do the above but separately for each index on axis 0; if more than 3 axes, the earliest axes will be ignored.

Caution

It does not completely guarantee that all unique sequences will be present in the output, as it relies on a hash and ignores collisions. If several sequences have the same hash, only one of them is kept, even if the actual content in the sequence is different.

Caution

Even if there are no repeated sequences, the output may be different from src. That is, new2old_indexes may NOT be an identity map even if nothing was removed.

Parameters

  • src (RaggedInt) – The input ragged tensor. Must have src.num_axes() == 2 or src_num_axes() == 3

  • need_num_repeats (bool) – If True, it also returns the number of repeats of each sequence.

  • need_new2old_indexes (bool) –

    If true, it returns an extra 1-D tensor new2old_indexes. If src has 2 axes, this tensor contains src_idx0; if src has 3 axes, this tensor contains src_idx01.

    Caution

    For repeated sublists, only one of them is kept. The choice of which one to keep is deterministic and is an implementation detail.

Returns

  • ans: A ragged tensor with the same number of axes as src and possibly fewer elements due to removing repeated sequences on the last axis (and with the last-but-one indexes possibly in a different order).

  • num_repeats: A tensor containing number of repeats of each returned sequence if need_num_repeats is True; it is None otherwise. If it is not None, num_repeats.num_axes() is always 2. If ans.num_axes() is 2, then num_repeats.dim0() == 1 and num_repeats.num_elements() == ans.dim0(). If ans.num_axes() is 3, then num_repeats.dim0() == ans.dim0() and num_repeats.num_elements() == ans.tot_size(1).

  • new2old_indexes: A 1-D tensor whose i-th element specifies the input sublist that the i-th output sublist corresponds to.

Return type

Returns a tuple containing

regular_ragged_shape¶

k2.ragged.regular_ragged_shape(dim0, dim1)[source]¶

Returns a RaggedShape with dim0() == dim0 and tot_size(1) == dim0 * dim1.

Require dim0 >= 0 and dim1 >= 0.

Caution

The returned ragged shape is on CPU. You can use its to method to move it to GPU.

Parameters

  • dim0 (int) – It equals to ans.dim0()

  • dim1 (int) – It equals to ans.tot_size(1) / dim0

Return type

RaggedShape

Returns

Return a ragged shape with 2 axes.
